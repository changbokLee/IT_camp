Links

Additional modules and tools for Cerberus X can be found at the following places:

Modules

  • Box2d - a Cerberus X port of the popular 2d physics library.
  • Diddy - a collection of utilties and extensions for Cerberus X.
  • Font machine - a very useful bitmap font module.

    Frameworks

    • Flixel - a port of the popular Flash framework.
    • Ignition - a new game framework for Cerberus X by Playniax.

      Tools and Utilities

      • Jungle - an awesome IDE for Cerberus X by LemonBytes.
      • https://www.cerberus-x.com/community/ - the place for Cerberus X folks
