Keyword Enumerate

Declares an enumeration of constant integer values.

Syntax

Description

Enumerate Identifier, Identifier, Identifier, etc.

Enumerate Identifier = Startvalue, Identifier, Identifier, etc.

Description

The Enumerate keyword declares an enumeration of constant integer values which start with 0..

The alternative syntax shows how to add a different start value to an element.
