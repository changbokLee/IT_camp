Interface IPrettifier

Declarations

Methods
BeginPrettyBlock : String () This method is called when a Markdown objects encounters a <pre> tag, and should return an appropriate html block start tag, eg: <div>.
EndPrettyBlock : String () This method is called when a Markdown object encounters a </pre> tag, and should return an appropriate html block end tag, eg: </div>.
PrettifyLine : String ( text:String ) This method is called by a Markdown object for every line inside a <pre> block, and should returned an html 'prettified' version of text.
