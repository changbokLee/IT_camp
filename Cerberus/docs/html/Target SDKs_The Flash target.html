The Flash target

Flash is a very popular web based platform by Adobe.

You will need

  • Flex SDK - open source compiler tools from Adobe/Apache.
    • Java SE JDK - the Java Development Kit.

      Notes

      The FLEX_PATH var in your Trans config file must be set correctly.

      To modify the size and layout of Flash Cerberus X apps, you will need to modify the CerberusGame.as file in your app's .build/flash directory.
